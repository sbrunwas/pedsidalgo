# Design Brief For Codex

## Style
- Clean, calm pediatric clinical decision support.

## Aesthetic
- CHOP pathway meets modern health tech UI.

## Typography
- Headings: `Inter` or `DM Sans`
- Body: `Inter`
- Data/metrics: `IBM Plex Mono`

## Color Palette
- Teal primary: `#2FA7A0`
- Navy secondary: `#1F2A44`
- Coral warning: `#F06A6A`
- Light gray background: `#F5F7FA`
- White cards
- Amber Kawasaki highlight: `#F5C04A`

## Layout
- Left input column.
- Right differential panel.
- Card-based expandable diagnoses.
- Rounded corners (`8px`).
- Subtle shadows.
- Clean spacing.
- Minimalist medical icons.
- Collapse/expand disease cards.
- Color-coded suspicion levels.
- Dark mode optional.
